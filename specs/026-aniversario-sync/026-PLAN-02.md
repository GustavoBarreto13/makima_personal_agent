<task id="T-02-D">

  <action>
    Criar o arquivo agents/kaguya/komi_sync.py espelhando a estrutura de agents/kaguya/gcal_sync.py.
    Estrutura do módulo (em ordem):

    1. Docstring de módulo descrevendo propósito (espelho best-effort Komi↔Kaguya), uso de
       KOMI_SYNC_ENABLED, anti-loop por convergência de valor, lazy imports para evitar ciclo.

    2. Imports no topo: apenas os, logging, agents.db (run_select, run_dml, get_conn).
       NUNCA importar agents.komi.tools ou agents.kaguya.tools_tasks no topo.

    3. _enabled() -> bool:
       Lê os.environ.get("KOMI_SYNC_ENABLED", "true").lower() != "false".
       Docstring Google Style. Comentário: "Lê em cada chamada para permitir toggle em runtime."

    4. _derive_task_title(person_name: str, label: str) -> str:
       Se label.strip().lower() in ('aniversário', 'aniversario'): return f"Aniversário de {person_name}"
       Senão: capitaliza primeira letra do label + f" de {person_name}"
       Ex.: "aniversário de namoro" → "Aniversário de namoro de {nome}"

    5. push_person_date(date_id: int) -> None:
       Toda a função envolve try/except Exception: pass (silencia tudo).
       Guard _enabled() logo no início — retorna se False.
       Carrega person_date: SELECT pd.id, pd.label, pd.date, pd.recurring, pd.person_id, p.name
         FROM person_dates pd JOIN people p ON p.id = pd.person_id AND p.deleted = FALSE
         WHERE pd.id = %(date_id)s
       Se None ou label não contém 'anivers' (usar LOWER(label) LIKE '%%anivers%%'): no-op.
       Verifica link existente: SELECT task_id FROM birthday_sync_links WHERE person_date_id = %(pd_id)s
       Se link existe (anti-loop D-04):
         Carrega tarefa atual via SELECT title, due_date FROM tasks WHERE id = %(task_id)s
         Deriva título esperado via _derive_task_title; compara com tasks.title e tasks.due_date
         Se idênticos → no-op (convergência, sem write)
         Se diferente → lazy import: from agents.kaguya.tools_tasks import update_task
           Chama update_task(task_id, due_date=date.isoformat(), title=title_esperado)
       Se link não existe:
         Lazy import: from agents.kaguya.tools_tasks import create_task, _get_birthdays_list_id
         Obtém list_id abrindo cursor explícito para chamar _get_birthdays_list_id(cur)
         Chama create_task(title=..., type="birthday", due_date=...,
                           project_id=list_id, person_ids=[str(person_id)])
         O create_task retorna o new_id; gravar link com INSERT INTO birthday_sync_links
         (person_date_id, task_id, komi_label) VALUES (%s, %s, %s) ON CONFLICT DO UPDATE SET task_id=EXCLUDED.task_id

    6. remove_person_date(task_id: int) -> None:
       Guard _enabled(); try/except.
       Lazy import: from agents.kaguya.tools_tasks import delete_task
       Chama delete_task(task_id, scope="series").
       (O birthday_sync_links cascade já foi removido pelo ON DELETE CASCADE no person_date.)
       Comentário: "O task_id é recebido do caller que o leu ANTES de deletar a person_date."

    7. push_birthday(task_id: int) -> None:
       Guard _enabled(); try/except.
       Carrega tarefa: SELECT t.id, t.title, t.due_date, t.type, t.deleted_at,
         pl.person_id, p.name
         FROM tasks t
         LEFT JOIN person_links pl ON pl.entity_type = 'task' AND pl.entity_id::int = t.id
         LEFT JOIN people p ON p.id = pl.person_id AND p.deleted = FALSE
         WHERE t.id = %(task_id)s LIMIT 1
       Se None ou type != 'birthday' ou due_date IS NULL ou person_id IS NULL: no-op (log).
       Verifica link: SELECT person_date_id FROM birthday_sync_links WHERE task_id = %(task_id)s
       Se link existe (anti-loop):
         Carrega person_date atual: SELECT date FROM person_dates WHERE id = %(date_id)s
         Compara due_date com person_dates.date — se iguais: no-op
         Se diferente: lazy import agents.komi.tools.update_important_date
           Chama update_important_date(person_date_id, date=due_date.isoformat())
       Se link não existe:
         Lazy import agents.komi.tools.add_important_date
         Chama add_important_date(person_id=str(person_id), label="aniversário",
                                   date=due_date.isoformat(), recurring=True)
         Usa result["id"] como person_date_id; grava birthday_sync_links

    8. remove_birthday(task_id: int) -> None:
       Guard _enabled(); try/except.
       Busca person_date_id: SELECT person_date_id FROM birthday_sync_links WHERE task_id = %(task_id)s
       Se não existe: no-op.
       Lazy import: from agents.komi.tools import delete_important_date
       Chama delete_important_date(person_date_id).
       (Cascade remove o link junto com o delete da person_date.)
