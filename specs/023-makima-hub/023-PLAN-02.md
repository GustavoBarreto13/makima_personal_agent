Helpers internos (todos com docstring Google-style, comentários pt-BR):

- `def _stat(v: str, k: str) -> dict:` → retorna `{"v": v, "k": k}`. Centraliza o formato de um stat.
- `def _agent(stat: dict, stat2: dict) -> dict:` → retorna `{"stat": stat, "stat2": stat2}`. Formato de um agente (REQ-11).
- `def _trunc(title: str, n: int = 14) -> str:` → se `len(title) > n`, retorna `title[:n] + "…"`, senão `title`. Truncamento de títulos (Frieren/Mai).
- `def _fmt_brl(v: float) -> str:` → formata um valor em reais com separador de milhar PONTO e SEM casas decimais no padrão "R$ X.XXX", com sinal explícito `+`/`−` (usar o caractere unicode "−" U+2212 conforme exemplos do SPEC: "+R$ 2.480" / "−R$ 2.480"). Usar `abs()` para o número e prefixar o sinal. Para o separador de milhar: formatar com `f"{int(round(abs(v))):,}".replace(",", ".")`.

Cada agente é uma função privada `_nami() -> dict`, `_frieren() -> dict`, etc., que retorna `_agent(...)` e levanta exceção livremente (capturada no endpoint). Definição de cada uma:

**`_nami()` (REQ-12):**
- `mes_ini, mes_fim = primeiro e último dia do mês corrente` (computar de `date.today()`; primeiro = `today.replace(day=1)`, último via `calendar.monthrange`).
- Query 1 (saldo do mês): `SELECT COALESCE(SUM(CASE WHEN tipo='Receita' THEN valor ELSE 0 END),0) AS receitas, COALESCE(SUM(CASE WHEN tipo='Despesa' THEN valor ELSE 0 END),0) AS despesas FROM transactions WHERE data BETWEEN %(ini)s AND %(fim)s AND deleted = FALSE`. `saldo = float(receitas) - float(despesas)`. `stat.v = _fmt_brl(saldo)`, `stat.k = "saldo do mês"`.
- Query 2 (lançamentos 7 dias): `SELECT COUNT(*) AS n FROM transactions WHERE data >= CURRENT_DATE - 7 AND deleted = FALSE`. `stat2.v = str(n)`, `stat2.k = "lançamentos / semana"`.

**`_frieren()` (REQ-13):**
- Query 1 (lendo agora): `SELECT b.title, b.total_pages, COALESCE(MAX(rl.page_end),0) AS current_page FROM books b LEFT JOIN reading_logs rl ON rl.book_id = b.id WHERE b.status = 'lendo' AND b.deleted = FALSE GROUP BY b.id, b.title, b.total_pages ORDER BY b.updated_at DESC LIMIT 1`. (Confirmar nome da coluna soft-delete em books: se for `deleted` BOOLEAN, usar `deleted = FALSE`; se a tabela não tiver, omitir a cláusula — verificar schema_pg.sql antes de escrever).
  - Se nenhuma linha → `stat.v = "—"`.
  - Senão: `pct = round(current_page * 100 / total_pages)` quando `total_pages` truthy, senão 0. `stat.v = f"{_trunc(title)} · {pct}%"`.
  - `stat.k = "lendo agora"`.
- Query 2 (lidos no ano): `SELECT COUNT(*) AS n FROM books WHERE status = 'lido' AND deleted = FALSE AND EXTRACT(YEAR FROM updated_at) = EXTRACT(YEAR FROM CURRENT_DATE)`. `stat2.v = str(n)`, `stat2.k = "livros este ano"`.

**`_komi()` (REQ-14):**
